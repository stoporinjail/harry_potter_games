# Harry Potter Games

---
Hi, welcome to the Harry Potter games.
# Requirements to run:
[Python](python.org) (3.7 or later)
[Git](https://git-scm.com/downloads)

How to run:
# Clone this repo
```
$ git clone https://github.com/stoporinjail/harry_potter_games
```
```
$ cd harry_potter_games
```
# Run the launcher file:
```
$ python launcher.py
```
